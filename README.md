# Azure Enterprise Landing Zone (Free-Tier Edition)

This project implements a production-style Azure Landing Zone using:

- Terraform
- GitHub Actions
- Hub-and-spoke network architecture
- Governance (management groups, policies, RBAC)
- Monitoring, backup, and DR design

## Status

- Day 1: Repository and folder structure created.

## High-Level Goals

- Use only free or very low-cost Azure services.
- Showcase enterprise-style architecture and workflows.
- Use Terraform and CI/CD to manage everything as code.
